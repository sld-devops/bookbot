# 🤖 BookBot

A text analysis tool built while completing the **Learn Python** course on [Boot.dev](https://boot.dev).

## 📚 About

BookBot reads a given book file and generates a statistical report — counting total words and the frequency of every letter in the text.

## 🛠️ What I Learned

- Reading and processing files in Python
- Working with strings and dictionaries
- Sorting and formatting output data
- Structuring a Python project across multiple files
- Writing clean, readable functions

## 📁 Structure

| File/Folder | Description |
|---|---|
| `main.py` | Entry point — runs the analysis and prints the report |
| `stats.py` | Helper functions for counting words and letters |
| `books/` | Directory containing book text files to analyse |

## 🚀 Usage
```bash
python main.py
```
